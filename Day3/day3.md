# Day 3 - Role-Based Prompting

## Objective

Learn how assigning different roles changes the quality and focus of AI responses.

## Question Used

How can I build a successful AI learning portfolio as a student?

## Without Role Prompt

Prompt:
How can I build a successful AI learning portfolio as a student?

Observation:
Claude provided a general answer suitable for most users.

## Founder Role Prompt

Prompt:
Act as a startup founder.

How can I build a successful AI learning portfolio as a student?

Observation:
The response focused on entrepreneurship, networking, and personal branding.

## Developer Role Prompt

Prompt:
Act as a senior software developer.

How can I build a successful AI learning portfolio as a student?

Observation:
The response focused on technical projects, GitHub, coding skills, and software development.

## Key Learnings

* AI changes its response based on the assigned role.
* Role-Based Prompting produces more relevant answers.
* Different roles provide different perspectives.
* Better prompts lead to better outputs.

## Comparison

| Prompt Type | Focus                       |
| ----------- | --------------------------- |
| No Role     | General Advice              |
| Founder     | Business & Growth           |
| Developer   | Technical Skills & Projects |
