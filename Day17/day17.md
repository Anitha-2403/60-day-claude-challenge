# Day 17 – Vehicle Cost Analysis Dashboard using Claude AI

## 🎯 Objective

The objective of this task was to use Claude AI to analyze a vehicle dataset (CSV) and generate a fully responsive HTML dashboard that visualizes vehicle operating costs, fuel comparisons, maintenance expenses, and environmental impact using interactive charts and KPIs.

---

## 🚗 Vehicle Details

- **Vehicle Model:** Honda Activa 6G
- **Fuel Type:** Petrol
- **Usage:** City
- **Distance per Month:** 1000 km
- **Vehicle Age:** 3 Years

---

## 📊 Dashboard Features

The generated dashboard includes:

- Vehicle Information Header
- KPI Cards
- Average Cost per Kilometer
- Monthly Fuel Cost
- Break-even Price Analysis
- Fuel Cost Comparison
- CO₂ Emission Analysis
- Maintenance Cost Analysis
- Cost vs Vehicle Age
- E85 Economics
- E85 Score Gauge
- Fuel Comparison Cards
- Key Takeaways Section

---

## 📈 Data Analysis Performed

The dashboard calculated and analyzed:

- Average Fuel Cost per Kilometer
- Average CO₂ Emissions per Kilometer
- Average Maintenance Cost per Kilometer
- Average Refuel/Recharging Time
- Vehicle Age-Based Cost Analysis
- E85 Cost Comparison
- Break-even Price Calculation
- Monthly Fuel Expense Estimation
- Environmental Impact Comparison
