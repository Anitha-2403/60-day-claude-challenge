# 📄 Day 39 – PDF Splitter & Merger

## 📌 Overview

As part of the **#60DayClaudeChallenge**, I built a **PDF Splitter & Merger** using **Claude AI**, **HTML**, **CSS**, and **JavaScript**. 
The application processes PDF files entirely in the browser, allowing users to split and merge documents securely without a backend.

---

## 🎯 Objectives

- Build a modern PDF management tool.
- Split and merge PDF files using client-side processing.
- Create a responsive and user-friendly interface.
- Deliver a smooth and interactive user experience.

---

## 🚀 Features

### 📄 PDF Splitter
- Upload and preview PDF files
- Split by page number or custom page ranges
- Extract selected pages
- Page thumbnails and validation

### 📚 PDF Merger
- Upload multiple PDF files
- Drag-and-drop file reordering
- Merge and download PDFs
- File preview with page count

### 🎨 UI/UX
- Responsive design
- Dark theme
- Smooth animations
- Processing indicators
- Success notifications
