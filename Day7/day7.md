# Day 7 - Claude Usage Strategy

## Objective

Learn how to use different Claude models and effort settings effectively based on personal goals and daily activities.

## My Profile

### Current Situation

Student (B.Tech AIML Student)

### Primary Activities

* Coding
* Learning
* Research
* Career Preparation
* Resume Building
* AI Projects
* Data Analytics

### Claude Usage

Daily User

### Most Needed Outputs

* Learning Support
* Coding Help
* Deep Research
* Career Guidance

## Recommended Primary Model

Claude Sonnet

## Why This Model Fits Me

Claude Sonnet provides the best balance of speed, reasoning, coding assistance, learning support, and project guidance. It is ideal for students preparing for AI, Data Analytics, and Software Development careers.

## When to Use Haiku

* Quick questions
* Simple explanations
* Fast summaries
* Basic brainstorming

## When to Use Sonnet

* Coding help
* Learning AI concepts
* Project development
* Interview preparation
* Resume optimization
* GitHub documentation

## When to Use Opus

* Deep research
* Complex project planning
* Advanced reasoning tasks
* Career roadmap generation

## Recommended Effort Levels

### Low

Quick answers and basic questions.

### Standard

Daily learning, coding, and project work.

### High

Research, project planning, and interview preparation.

### Max

Complex reasoning, career strategy, and detailed reports.

## My Personalized Claude Workflow

| Task            | Best Model | Best Effort | Reason                      |
| --------------- | ---------- | ----------- | --------------------------- |
| Coding          | Sonnet     | Standard    | Balanced speed and accuracy |
| Learning AI     | Sonnet     | Standard    | Strong explanations         |
| Research        | Opus       | High        | Deep analysis               |
| Resume Building | Sonnet     | High        | Better optimization         |
| Career Planning | Opus       | Max         | Advanced reasoning          |
| Quick Questions | Haiku      | Low         | Fast responses              |

## Biggest Mistakes To Avoid

* Using high effort for simple tasks.
* Asking vague questions without context.
* Not using the appropriate model for the task.
* Ignoring Claude's reasoning process.

## Biggest Insight

Different Claude models serve different purposes. Choosing the correct model and effort level can significantly improve productivity and output quality.

## Conclusion

Claude Sonnet with Standard Effort is the best choice for most of my daily learning, coding, and career preparation activities.
