# Day 18 – Brain Dump Action Planner using Claude AI

## 🎯 Objective

The objective of this task was to create a reusable Claude Custom Skill that transforms unstructured meeting notes, brainstorming sessions, voice transcripts, and project discussions into a structured, interactive HTML dashboard. The dashboard organizes information into summaries, action items, risks, blockers, open questions, and decisions without adding or modifying the original content.

---

## 🛠️ Custom Skill Details

**Skill Name:**
brain-dump-action-planner

**Description:**
Transform messy notes, meeting transcripts, voice memos, brainstorming sessions, and stream-of-consciousness thoughts into structured summaries, action plans, decisions, open questions, and task lists while preserving all original information.

---

## 📋 Sample Input Used

I tested the custom skill using professional project meeting notes for an AI Product Development project.

The meeting included:

- Project progress updates
- Team responsibilities
- Technical discussions
- Action items
- Risks and blockers
- Open questions
- Decisions
- Next meeting schedule
