# Day 8 - Personal Environmental Health Analyzer

## Objective

Learn how AI can generate complete interactive dashboards using Context Engineering and Data Analysis.

## Project

Created a Personal Environmental Health Analyzer dashboard using Claude.

The dashboard analyzes environmental conditions and presents air quality, pollution metrics, environmental health scores, and personalized recommendations through an interactive interface.

## Features

### Key Metrics

* Average AQI
* Highest AQI City
* Lowest AQI City
* Environmental Health Score
* Number of Cities Analyzed

### Interactive Visualizations

* AQI Comparison Chart
* PM2.5 Comparison Chart
* PM10 Comparison Chart
* AQI Distribution Chart
* City Ranking Chart

### Interactive Filters

* City Selector
* AQI Range Filter
* Pollutant Filter
* Health Risk Filter
* City Comparison Mode

### Personal Report Card

* Air Quality Score
* Water Quality Score
* Environmental Health Score
* Air Quality Grade
* Water Quality Grade
* Hair Risk Indicator
* Skin Risk Indicator

### Insights

* Cleanest Cities
* Most Polluted Cities
* Biggest Anomaly
* Most Surprising Observation
* Recommended Actions

## Key Learnings

* AI can generate complete dashboard applications.
* Environmental data can be analyzed visually.
* Interactive dashboards improve data understanding.
* Context-rich prompts create significantly better outputs.

## Biggest Insight

Providing detailed requirements helped Claude generate a complete, professional dashboard instead of a simple report.

## Conclusion

This project demonstrated how AI can combine data analysis, visualization, UX design, and frontend development into a single interactive application.
